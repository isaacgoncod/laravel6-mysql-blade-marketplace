Opções para o ON DELETE e para o ON UPDATE


cascade: remove em cascata todos os registros filhos associados a esta loja (tabela);

restrict: impede qualquer alteração no dado pai(loja) enquanto tiverem filhos(produtos);

no action: opção padrão e que lança o erro recebido;

set null: na remoção do pai seta os filhos como null, na chave referência;

set default: na remoção do pai seta uma valor padrão para a referência nos filhos.
